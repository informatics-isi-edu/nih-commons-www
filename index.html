--- layout: default title: NIH Commons DERIVA Demo ---

Goal of the demo: To demonstrate faceted search, browse capability and BDBag export capabilities.

Users should be able to:

  • Search and browse data in the repository.
  • Export a selected set of data.

Datasets presented in the demo:

  • GTEx public data set
  • EBI data
  • Combined GTEX/EBI

Dataset Limitations:

  • GTEx datasets are not able to include CRAI files in the Bag export because no file length or md5 values were provided by the GTEx data stewards.
  • EBI datasets have sequencing file information, but not the metadata associated with samples.
