--- layout: default title: NIH Commons DERIVA Demo ---

Goal of the demo: To demonstrate faceted search and browse capability and BDBag export capabilities.

Users should be able to:

  • search and browse data in the repository
  • export selected set of data

Datasets presented in the demo:

  • GTEx public data set
  • EBI data
  • Combined GTEX/EBI

Dataset Limitations:

  • GTEx data sets are not able to include CRAI files in Bag export because no file length or md5 values were provided by GTEx data stewards.
  • EBI data sets have sequencing file information, but not the metadata associated with samples.
